# 🚛 Simulador de Estabilidade de Carga em Caminhão

## 📍 Contexto real
Na operação logística, a distribuição incorreta de carga em caminhões pode gerar:

- risco de tombamento
- sobrecarga em eixos
- multas por irregularidade
- aumento de desgaste do veículo
- perda de eficiência operacional

Pensando nisso, desenvolvi uma simulação visual para análise de estabilidade de carga.

---

## 💡 O que este projeto faz
Este projeto simula o **comportamento da carga em um caminhão**, permitindo:

- adicionar e remover cargas
- visualizar distribuição no compartimento
- calcular impacto nos eixos
- analisar inclinação lateral

👉 Tudo isso em tempo real.

## ⚙️ Funcionalidades
